#  Campus Lost & Found Portal

A simple web-based platform that helps students report, search, and recover lost or found items within the campus.

---

##  Project Overview

Campus Lost & Found Portal is a lightweight frontend web application designed for students to post lost or found items, browse listings, and connect with rightful owners. It uses basic web technologies and browser storage to simulate real-world functionality.

---

##  Features

-  User Registration & Login (basic)
-  Post Lost / Found Items
-  Upload item details with description and image
-  Browse and search items
-  Claim request system
-  Admin panel to manage posts
-  Data stored using localStorage (no backend)

---

##  Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Browser LocalStorage
